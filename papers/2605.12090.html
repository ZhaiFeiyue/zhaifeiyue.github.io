World Action Models 综述 (2605.12090) — L3 per-paper synthesis #

Cluster 说明：本篇 target 是 embodied-AI / robot-learning 领域的分类综述，而它的 8 个 related peers 是同属 category: algorithm 但主题高度异质的一组（attention 量化 ×2、GPU DMA offload、massive-value 可解释性、long-context benchmark、omni-modal 训练、agentic LLM）。 唯一在主题上真正同源的是 DreamerV3 [2301.04104]（world model / model-based RL）。 因此本 L3 采取"1 强相关 + 7 弱/工具相关"的诚实定位：对 DreamerV3 做深度 delta，对其余 peers 只抽取可迁移的机制/方法论类比，并明确标注哪些是 category-level 而非 topic-level 相关。

1. 相关论文 #

按与 target 的相关强度分三档：

A 档 · 主题同源（world model 谱系） #

  • DreamerV3 (2301.04104) — 唯一真正的"世界模型"同宗。它是 model-based RL 的 RSSM 世界模型

    • + 想象空间 actor-critic [2301.04104]，与 WAM 综述里 Cascaded 分支的

    "先学 latent 世界、再在其上做策略"完全同构。DreamerV3 恰是 WAM 综述 §4（Fig 4）所称

    "WM 作为外生工具服务策略学习"的经典 RL 实例：在想象轨迹里训 actor 而不触真实环境

    [2301.04104]。综述把 WAM 与它区分开的判据是"耦合动作生成"——DreamerV3 的

    actor 消费世界模型的 latent，但世界模型本身是 action-conditioned $p(o'\mid o,a)$，属于

    WM+外置策略，而非综述定义的 joint $p(o',a\mid o,l)$ [2605.12090]。

B 档 · 方法论可迁移（机制/训练范式类比，非同主题） #

  • Massive Values (2502.01563) — 揭示 RoPE 低频维度的 massive values 专职上下文理解

    • [2502.01563]。这与 WAM 的"哪些内部表示真正承载物理前瞻"是同一类

    机制归因问题：WAM 综述提出"推理时可丢弃未来预测头而不掉性能"的反直觉规律

    [2605.12090]，正需要 2502.01563 式的 disruption / 归因方法学去证伪。

  • Kimi K2 (2507.20534) — agentic 数据合成 + stateful tool-simulator world model

    • [2507.20534]。K2 的"用 LLM 模拟有状态工具环境来合成轨迹"与 WAM 的

    "用学习到的世界模型做 rollout / 生成训练轨迹"是同一想象-合成范式在 LLM-agent 与

    embodied 两个域的镜像。

  • DreamerV3 亦跨档：其"固定超参跨域鲁棒性技巧"[2301.04104] 对 WAM 综述

    • §7 呼吁的"跨耦合机制对照实验"提供了方法论范式（量纲无关变换让公平对照成为可能）。

C 档 · 工具/基座相关（会被 WAM 系统使用，但非概念相关） #

  • SageAttention (2410.02367) / SageAttention3 (2505.11594) — attention 量化加速

    • [2410.02367][2505.11594]。二者是视频扩散/DiT 骨干的推理加速器；

    而 WAM 的 Joint-Diffusion 分支高度依赖 Wan2.x / Cosmos / CogVideoX 等视频扩散基座

    [2605.12090]，Sage 系正是这些基座落地时缓解"延迟税"的现成工具。

  • DMA offload (2412.14335) — GPU 上 compute-communication overlap [2412.14335]，

    • 与 WAM 属集群/系统层，仅在"部署 WAM 大模型"时相关。

  • Qwen2.5-Omni (2503.20215) — omni-modal Thinker-Talker 流式生成 [2503.20215]，

    • 其"多模态时间对齐 TMRoPE"与 WAM 的视频-动作时间对齐是弱类比。

  • NoLiMa (2502.05167) — long-context 评测方法论 [2502.05167]，与 WAM 综述

    • §6 的"评测解耦"批评是评测哲学上的呼应（见 §3、§5）。

2. 本篇 vs 相关论文的 delta #

Δ1 vs DreamerV3：从"action-conditioned WM + 外置策略"到"joint $p(o',a\mid o,l)$" #

DreamerV3 学的世界模型是 $p(o'\mid o,a)$（给定动作预测未来），策略 actor 是在其想象 latent 上

分离训练的 [2301.04104]。这恰好落在 WAM 综述的 Cascaded 象限——

"先建世界、再解动作"，训练解耦 [2605.12090]。综述的新增贡献是把这一类

与 Joint（世界与动作联合建模、联合训练、可吞无动作视频）并列成正交坐标，并指出 Joint 的独特

价值是"统一数据消化"——DreamerV3 无法直接吃互联网无动作视频，因为它的世界模型是 action-conditioned

[2605.12090]。Delta 本质：DreamerV3 是"闭环 RL 里的世界模型"，WAM 综述

把视野扩到"用互联网视频监督的 open-loop 生成式策略"，并把 DreamerV3 收编为一个特例。

Δ2 vs Massive Values / DreamerV3：机制归因方法学的缺口 #

2502.01563 用 prefilling-only disruption 精确证明"massive values = 上下文理解通道"

[2502.01563]；DreamerV3 用 14 任务消融证明"性能主要来自世界模型的无监督重构

信号而非奖励梯度"[2301.04104]。二者都提供了 WAM 综述缺失的东西——

对"世界建模到底贡献了什么"的因果隔离。综述只观察到"推理时丢弃未来预测头不掉分"这一现象

[2605.12090]，却没有 disruption 式实验去确认收益来自训练期辅助梯度

而非推理期前瞻。Delta：综述提出了假说，2502.01563/2301.04104 提供了验证它的方法模板。

Δ3 vs NoLiMa：评测解耦批评的"同构"但更弱 #

WAM 综述的核心诊断之一是"世界建模用 PSNR/FVD、动作用成功率，二者缺因果一致性联合指标"

[2605.12090]。NoLiMa 对 long-context 做了完全平行的诊断——现有 NIAH 靠表面

词面匹配虚高，去掉后性能崩塌 [2502.05167]。Delta：NoLiMa 不止批评，还

造了新 benchmark（ROUGE R-1=0.069）并给出 13 模型实测；WAM 综述只指出缺口、引用 Wow-wo-val!

的 IDM Turing Test 作为锚点 [2605.12090]，尚未产出统一评测协议。综述在"诊断"上

到位，在"补救"上落后于 NoLiMa 这种单点评测工作。

Δ4 vs Sage 系 / DMA / Omni：WAM 的"延迟税"是这些工具的用武之地 #

综述给出关键数字：DreamZero 经算法+CUDA 优化仍只到 ~7Hz，落后非生成式 VLA 的 50Hz 标准一个

数量级 [2605.12090]。SageAttention3 在 Blackwell 上把 DiT attention 推到

1038 TOPS（5× FA2）[2505.11594]，Qwen2.5-Omni 用 sliding-window DiT + block-wise

encoder 把首包延迟压到 ~640ms [2503.20215]。Delta：这些是 WAM 综述

没有覆盖但直接相关的系统层杠杆——综述把"推理延迟"列为开放挑战却停在算法层，未讨论量化/

流式/DMA 这些正交加速路径。

3. 可攻击面 #

攻击 1：判据的"可判定性"存在灰区，DreamerV3 的归类可被质疑 #

综述用"前向预测 + 耦合动作"两判据把 DreamerV3 类归为 Cascaded [2605.12090]。

但 DreamerV3 的 actor 完全在世界模型想象的 latent 上训练、从不触真实环境

[2301.04104]——这种"动作与预期未来在同一 latent 空间深度耦合"其实比综述描述的

"分离训练 Cascaded"更接近 Joint 的精神。攻击点：综述自己承认判据"依赖架构解读，存在灰区"

[2605.12090]，DreamerV3 恰是骑墙样例——它训练上解耦（世界模型先学）、

表示上高度耦合（同一 RSSM latent）。二分骨架在此类样例上会强行归档，掩盖连续谱。

攻击 2："推理时丢弃未来预测头不掉分"缺因果证明 #

综述把这条当作跨方法归纳的反直觉规律 [2605.12090]，但它是观察性

的，没有 2502.01563 式的 controlled disruption。反驳：若真如此，可能意味着未来预测在这些模型里

只是训练期正则而非功能核心——但也可能是这些特定模型的世界分支本就欠拟合。借用 2502.01563 的

方法学 [2502.01563]：应做"训练时禁用世界分支 vs 训练时启用但推理时

丢弃"的对照。综述没做，规律成色存疑。

攻击 3：分类覆盖度是快照，且"无其它桶"可能是幸存者偏差 #

综述称 Table 1/2/3 把 50+ 模型全钉入分类、无"无法归类"桶 [2605.12090]。

但一个不留残差的分类，要么真的正交完备，要么是为了骨架自洽而牺牲了边界样例。对比 K2 的

tool-simulator world model [2507.20534]——它是"用 LLM 做世界模型来合成 agent

轨迹"，既非像素世界也非机器人 latent，综述的具身坐标系里根本没有它的位置。攻击点：一旦把

LLM-agent 世界模型纳入，"无残差"立刻破裂，说明覆盖度是领域窄化的产物。

攻击 4：延迟税数字是单点，缺横向公平对照 #

7Hz vs 50Hz 的"一个数量级延迟税"[2605.12090] 来自 DreamZero 单点，未控制

硬件/模型规模/量化状态。Sage 系证明同一 attention 在不同 bit-width / GPU 上吞吐可差 5×

[2505.11594]。反驳：延迟税可能大半是"未优化生成式基座"而非"生成式范式

的本质代价"——综述把它当范式级论断，证据强度不够。

4. 生态位 #

范式定位：综述型"坐标系工程"，不是模型贡献 #

本篇的生态位与其余 8 篇根本不同：peers 大多是单点技术贡献（一个 kernel、一个优化器、一个

benchmark、一个架构），而 target 是领域收敛器——把 366 篇碎片文献压成两判据 + Cascaded/Joint

正交骨架 [2605.12090]。它在生态里的角色类似 NoLiMa 之于 long-context 评测

[2502.05167]：不解决问题，而是重定义问题的坐标。

采纳证据：术语去混淆 + Awesome-WAM 清单 #

综述维护 openmoss.github.io/Awesome-WAM 与 GitHub 文献库 [2605.12090]，

这是综述类工作最典型的"活体采纳"信号（持续更新的社区清单）。相比之下 Sage 系的采纳证据是

pip install + ComfyUI/diffusers 集成 [2410.02367]，K2 是 HF

checkpoint 下载 [2507.20534]——都是可执行制品；而综述的采纳是

术语与分类被引用，属于软性但影响面更广的范式采纳。

paradigm-shift 判断 #

WAM 综述押注的范式转移是"从反应式 VLA $p(a\mid o,l)$ → 前瞻式 joint $p(o',a\mid o,l)$"

[2605.12090]。这一转移与 DreamerV3 十年前证明的"model-based 想象训练可行"

[2301.04104] 一脉相承，但把监督源从"RL 交互"换成"互联网视频"。生态位风险：

若延迟税（§3 攻击 4）与评测解耦（Δ3）不解决，Joint-WAM 可能停在"训练期有用、推理期被丢弃"的

半吊子状态（综述自己观察到的 §7 现象），沦为 VLA 的辅助训练技巧而非独立范式。

5. 未探索方向 #

从这个异质 cluster 里能抽出的 hybrid / adaptive 方向：

方向 1：借 disruption 方法学做 WAM 的"世界分支因果审计"（hybrid: 2502.01563 × WAM） #

把 2502.01563 的 prefilling-only disruption [2502.01563] 移植到

WAM：在推理时选择性破坏世界预测分支的中间表示，量化下游动作成功率的因果依赖。这能直接检验综述

悬而未决的"丢弃未来头不掉分"假说 [2605.12090]，把观察性规律升级为

因果结论。技术上完全可做，目前无人做。

方向 2：因果一致性联合指标 = NoLiMa 式"去表面线索"评测（hybrid: 2502.05167 × WAM §6） #

综述指出评测缺"想象未来与动作因果一致性"的联合指标 [2605.12090]。NoLiMa 的

洞察是"去掉表面捷径才能测真能力"[2502.05167]。可造一个 WAM benchmark：让

像素上"看起来对"但物理上违规（悬浮物体/反重力）的想象无法通过下游控制——即用 IDM 可执行性

作为过滤器，去掉"视觉可信但物理错误"这个捷径。这正是综述引用的 Wow-wo-val! IDM Turing Test

[2605.12090] 的系统化版本。

方向 3：自适应"训练用世界头、推理丢世界头"的量化协同（hybrid: Sage 系 × WAM 延迟税） #

综述观察到多个模型推理时丢弃世界分支 [2605.12090]，Sage 系证明推理侧

可用 FP4/INT8 大幅提速 [2505.11594]。自适应方向：训练期全精度双分支（世界+动作），

推理期不仅丢世界头、还对保留的动作分支做 attention 量化——两个正交的"减负"叠加，量化攻克延迟税

的另一半。综述把延迟当纯算法问题，忽略了这条系统路径。

方向 4：LLM-agent 世界模型 ↔ embodied 世界模型的统一坐标（hybrid: K2 × WAM 骨架） #

K2 的 stateful tool-simulator [2507.20534] 与 WAM 的物理世界模型在"用想象环境

合成训练轨迹"上同构，但活在两套互不相交的分类里。未探索方向：把综述的 Cascaded/Joint 判据

[2605.12090] 抽象到 modality-agnostic 层——"文本工具状态"与"像素物理状态"

只是 $o$ 的不同实例。这能修补 §3 攻击 3 指出的覆盖度窄化，产出跨 embodied/agentic 的统一 world-model

分类学。
