Summary

Core Contribution

AMD首个多芯片模块(MCM)封装GPU架构CDNA 2，通过双GCD封装和CPU-GPU缓存一致性驱动全球首台Exascale超算Frontier。

Summary

CDNA 2是AMD在计算GPU领域的重大跨越，首次采用多芯片模块(MCM)封装将两个GCD(Graphics Compute Die)通过interposer bridge以400GB/s双向带宽互联在同一封装内。每个GCD包含110个CU，双GCD合计220CU，配合FP64 Matrix Core的引入，实现了47.9 TFLOPS FP64 Vector和95.7 TFLOPS FP64 Matrix的惊人HPC性能，较MI100的11.5 TFLOPS实现4.2x飞跃。

显存升级至128GB HBM2e(3.2TB/s)，L2 Cache扩大至16MB(8MB/GCD)。外部8条Infinity Fabric链路提供800GB/s互联带宽。关键创新包括与EPYC处理器的硬件级CPU-GPU缓存一致性、向量单元Packed FP32支持(有效翻倍FP32向量吞吐)、以及L2 FP64原子操作增强。

CDNA 2驱动了Frontier——世界首台Exascale超算，由9,408个MI250X节点组成，峰值超过1.1 ExaFLOPS FP64。

Key Findings

  • MCM封装: 双GCD通过interposer bridge互联，400GB/s封装内带宽
  • FP64 Matrix Core: MFMA首次支持FP64矩阵运算，95.7 TFLOPS FP64 Matrix
  • 220 CU (110/GCD), 128GB HBM2e, 3.2TB/s
  • CPU-GPU缓存一致性: 与EPYC实现硬件级cache coherency
  • 800GB/s外部IF链路(8 links)，驱动Frontier Exascale超算

Limitations

  • 无FP8支持
  • 无结构化稀疏
  • 仍为7nm工艺，落后于同期NVIDIA 4N (H100)
  • 双GCD在操作系统层面呈现为2个逻辑GPU，编程模型有额外复杂性
  • 无Transformer Engine等AI专用加速

Infrastructure Impact

  • Algorithm: FP64 Matrix加速使科学计算矩阵运算全面受益
  • Kernel: Packed FP32需要编译器/调度器适配
  • Framework: ROCm + HIP持续追赶CUDA生态
  • System: MCM封装开创GPU多die路线，Frontier验证Exascale可行性
  • LLM: 128GB显存为早期大模型部署提供基础
Details

Deep Analysis (hardware)

1. Architecture Overview

  • 产品: AMD Instinct MI250X / CDNA 2
  • 工艺: TSMC 7nm
  • 封装: MCM — 2个GCD + interposer bridge
  • 目标市场: Exascale HPC与AI训练

2. Generational Delta (vs CDNA 1 / MI100)

新增: MCM双GCD封装, FP64 Matrix Core, Packed FP32, CPU-GPU缓存一致性, L2 FP64 atomic

移除: 无

增强: CU 220(vs 120), 显存128GB(vs 32GB), 带宽3.2TB/s(vs 1.23TB/s), IF链路800GB/s(vs 3x links)

3. Compute Unit Deep Dive

  • 每CU: 1个Matrix Core Engine (增加FP64 MFMA), 4个Vector单元(16-wide, 支持Packed FP32), Scalar单元
  • VRF: 128KB/CU, SRF: 3.2KB/CU
  • Wavefront: 64 work-items
  • MFMA支持: FP64/FP32/FP16/BF16/INT8输入

4. Peak Performance

5. Memory Subsystem

  • HBM2e: 128GB (64GB/GCD), 3.2TB/s (1.6TB/s/GCD)
  • L2: 16MB (8MB/GCD), 32 slices/GCD
  • Register: 128KB VRF/CU, 3.2KB SRF/CU

6. Interconnect

  • 封装内IF: 400GB/s双向 (interposer bridge)
  • 外部IF链路: 800GB/s (8 links)
  • PCIe Gen4: x16
  • CPU-GPU缓存一致性: 与EPYC处理器硬件级coherent

7. Software & ISA

  • ROCm (持续迭代)
  • MFMA指令族扩展: 新增FP64矩阵指令
  • Packed FP32: 向量单元在FP32操作中可打包执行
  • HIP + LLVM编译器
